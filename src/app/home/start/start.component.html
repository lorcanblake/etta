Get Started

The platform consists of two apps: 5ervice and 5ervice Business. 

git clone https://github.com/lorcanblake/5ervice.git
cd 5ervice
npm i
ng serve --o

git clone https://github.com/lorcanblake/5ervice-business.git
cd 5ervice-business
npm i
ng serve --o

Prototyping

Once you've grabbed Etta, prototyping is as easy as pie: 

<!DOCTYPE html>
<html lang="en">
<title> </title>
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="/path/to/etta.min.css">
<body>

</body>
</html>

Create a New Project

Etta is built in Angular 5. Clone it and poke around: 

git clone https://github.com/lorcanblake/etta.git
cd etta
npm i
ng serve --o
